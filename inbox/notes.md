1. Install solidity compiller
   ```bash
   npm install solc@0.4.17
   ```
2. Install `mocha`, `ganache`, and `web3` libraries
   ```bash
   npm install --save mocha ganache-cli web3
   ```
3. Install a wallet provider:
   ```bash
   npm install @truffle/hdwallet-provider
   ```
